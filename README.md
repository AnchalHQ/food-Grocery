# food-GroceryGrocery Store Management System 🛒

A Python-based CLI application to manage grocery inventory, calculate bills, and track stock levels. This project demonstrates CRUD operations and data handling in Python.

## ✨ Features
* **Inventory Management:** Add, update, and view grocery items.
* **Billing System:** Generate automated receipts with GST/Tax calculations.
* **Search Functionality:** Quickly find items by name or category.
* **Data Persistence:** (Optional) Saves data to a local file.

## 🛠️ Tech Stack
* **Language:** Python 3.x
* **Tools:** VS Code, Git
